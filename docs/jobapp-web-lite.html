Experimental Web App Lite

JobApp AI Assistant

Clean and structure a CV directly in the browser, then optionally use your own Gemini API key to tailor it to a job. The first practical target is Ciência Vitae: the official Portuguese CV platform widely used by researchers, institutions, and FCT-oriented funding/project workflows.

API-key warning:

This page runs in your browser and sends your request directly to Gemini using the key you paste here. PagBiOmicS does not store the key. Best practice: create a temporary or low-limit key, test the workflow, clear it from this page, and then revoke/delete it in Google AI Studio.

PDF parsing uses the PDF.js files vendored in this repository, not an external CDN.

Scope

What this Lite version can do

Good for quick browser tests

Heuristic parsing of PDF, TXT, Markdown, JSON, and XML CV files, with extra rules for structured Ciência Vitae exports.

Limited by the browser

No local database, no full Python parser, no desktop export workflow, and no reliable scraping when job boards block CORS or require login.

Desktop remains recommended

Use the Windows app for sensitive CVs, large Ciência Vitae exports, repeated applications, and stronger local parsing/exporting.
