d'BiYOK Lab

Bring Your Own Key for AI-assisted Apps

d'BiYOK Lab is Dany's idea to integrate Bi as Biodiscovery-Integrated with BYOK, Bring Your Own Key. The aim is a wider set of practical apps for professional productivity, research, learning, entrepreneurship, and everyday workflows.

The current public direction is simple: keep sensitive work local when possible, let users connect their own Gemini, OpenAI, Claude, Ollama, DeepSeek, GLM, or compatible provider, and make API use understandable instead of mysterious.

BYOK

1. JobApp AI Assistant

A local app for adapting structured CVs to job posts. It can parse, filter, match, and export editable application documents.

2. Local-first by default

Recommended for CVs, unpublished research, private client work, and personal API keys. The app runs on localhost.

3. BYOK with guidance

Users bring their own key or local model, with direct setup links and warnings about privacy, limits, and provider terms.

4. Wider app set

The same idea can support writing, data cleaning, literature work, business tasks, learning tools, and casual productivity.

No API key is collected on this page. The current recommendation is to enter keys only inside the local desktop app until a dedicated secure web mode exists.
