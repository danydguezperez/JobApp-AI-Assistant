JobApp AI Assistant

CV parser, job matcher, and editable application exporter.

A local-first assistant for adapting structured CVs to specific job opportunities. It helps users parse a CV, focus the most relevant evidence, match it against a role, and export editable application materials.

Download desktop app

Best for private CVs, local exports, longer documents, and repeated use. The Windows app runs on localhost and keeps files on the user's computer.

The Windows release is public on GitHub. Download the executable, run it locally, and use your own API key or a local model.

What JobApp AI Assistant does

Parse and structure CVs

Import PDF, TXT, or structured CV exports, then organize the content into editable English sections.

Focus the evidence

Select the experience, projects, publications, and skills that matter most for each job application.

Export editable outputs

Generate tailored CV bullets, cover letters, interview notes, and exportable Markdown, DOCX, PDF, or JSON files.
