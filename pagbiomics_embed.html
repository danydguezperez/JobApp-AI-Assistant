JobApp AI Assistant

CV parser, job matcher, and editable application exporter.

A local-first assistant for adapting structured CVs to job opportunities. It is especially useful for Ciência Vitae exports, the official Portuguese academic CV platform used by researchers and institutions in funding and project workflows such as FCT calls.

Choose the right mode

Try from the web: experimental Web Lite

API-key caution

Runs in the browser for quick BYOK tests. It can import PDF, TXT, Markdown, JSON, or XML and make a first heuristic pass over structured Ciência Vitae CVs.

Use a temporary or restricted API key, clear it after testing, and revoke/delete it in your provider account.

Open Web App Lite

Recommended: download desktop app

Best for repeated use, local exports, longer documents, structured Ciência Vitae CVs, and users who do not want to paste an API key into a browser page.

Runs on localhost, keeps files on the user's computer, and can use BYOK or local models.

What JobApp AI Assistant does

Parse and structure CVs

Import PDF, TXT, or structured Ciência Vitae exports, then organize the content into editable English sections without repeated headers or platform noise.

Focus the evidence

Select the experience, projects, publications, and skills that matter most for each job application.

Export editable outputs

Generate tailored CV bullets, cover letters, interview notes, and exportable Markdown, DOCX, PDF, or JSON files.
