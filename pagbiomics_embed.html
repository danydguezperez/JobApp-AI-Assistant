JobApp AI Assistant

A CV and job-application assistant designed to parse structured CVs, focus the evidence that matters for a specific role, and generate editable application materials.

Use from the web: JobApp Web Lite

Paste a CV/profile summary, paste a job description, add your own Gemini API key, and generate a first tailored application draft directly in the browser.

Your API key is used client-side in your browser for this Web Lite mode. PagBiOmicS does not collect it in this static embed. For sensitive CVs or long workflows, the desktop app remains the safer option.

How d'BiYOK Lab Works

Local-first when sensitive

CVs, unpublished research, client work, and private business documents should stay local whenever possible.

BYOK with guidance

Users can connect Gemini, OpenAI, Claude, Ollama, DeepSeek, GLM, or compatible providers depending on each app.

Productivity-driven

The lab is not limited to research. It can support writing, data cleaning, literature work, business tasks, learning, and everyday workflows.
